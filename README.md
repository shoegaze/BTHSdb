# BTHSdb
Django based database for Brooklyn Technical High School (BTHS)

ver. 1.0.0

Current Features:
-----------------
* Users
  - Log in menu
* Equipment database
  - Subjects
  - Rooms
  - Shelves
  - Nullable: 
    - Take out time
    - Take out user
